* LFE and OTP example

An OTP application can work with LFE code.
The secret was adding the name of LFE module to ~src/why.app.src~

hello is the name of LFE module that was added.
#+BEGIN_EXAMPLE
 {mod, { why_app, [hello]}},
#+END_EXAMPLE


** Build
#+BEGIN_EXAMPLE
$ rebar3 lfe compile
$ rebar3 compile
#+END_EXAMPLE

** REPL
#+BEGIN_EXAMPLE
$ rebar3 lfe repl
#+END_EXAMPLE

Lisp code
#+BEGIN_SRC lisp
(: hello hello-lfe)
#+END_SRC

#+BEGIN_EXAMPLE
Hello from LFE project
ok
#+END_EXAMPLE

** Erlang shell

#+BEGIN_EXAMPLE
$ rebar3 shell
#+END_EXAMPLE

Call the same function using Erlang syntax.

#+BEGIN_EXAMPLE
3> hello:hello().
Hello from LFE project
ok
4>
#+END_EXAMPLE
